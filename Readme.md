# Golang API REST with Frontend Typscript, MySQL, Docker && Nginx

- Install

  - Go
  - Docker
  - docker-compose

- Usage:

```bash
    git clone https://github.com/Sommysab/My_Diary.git backend

    cd backend

    sudo docker-compose up -d
```

## Api endpoint for backend:

- localhost:8010/api

## Api endpoint for frontend:

- localhost:3000
